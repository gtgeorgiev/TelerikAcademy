#   C# Fundamentals Course Repository

Repository for the course [C# Fundamentals] at [Telerik Academy]

The course "C# Fundamentals" introduces to the fundamentals of programming. The course focuses on fundamental programming concepts using C#/.NET and Visual Studio. The course covers topics such as **data types**, **variables**, **operators**, **arithmetic expressions**, **Console Input/Output**, **conditional statements** (`if-else`, `switch-case`) and **loops** (`while`, `do-while`, `for`, `foreach`).
